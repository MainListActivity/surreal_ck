# OIP-10 — 首页可信指标与真实筛选语义

## Parent

`.scratch/operating-iteration-plan/PRD.md`

## What to build

移除首页固定在线人数和仍指向未完成功能的筛选语义，改为展示当前工作区真实成员数或最近七天活动数。真实指标不可用时隐藏整个指标，不用零值或模拟值制造协作活跃假象；工作簿标签只保留已经可操作的“全部、已固定、最近打开”等语义。

## Acceptance criteria

- [ ] 首页不再出现固定“4 位协作者在线”或其他模拟协作数据。
- [ ] 首页从现有成员或活动数据源展示至少一个真实工作区指标。
- [ ] 指标请求失败或当前身份不可读时隐藏指标，页面其余功能正常。
- [ ] “共享工作簿”等尚无真实权限语义的入口被删除或替换为已有功能。
- [ ] 工作簿筛选的选中态、空态和结果与标签文案一致。
- [ ] 测试明确禁止模拟在线人数回归，并覆盖有数据、无数据和失败状态。

## Blocked by

None - can start immediately
